 1. TODO (Быстрая заметка)
   * Цель: Мгновенная фиксация. "Сделай и забудь".
   * Пользователь: Ты (быстро набросал) -> Агент (сделал).
   * Жизненный цикл: Короткий (Часы).


2. ANALYSIS (Расследование)
  * Цель: Поиск истины. "Почему сломалось?" / "Как это работает сейчас?".
  * Пользователь: Агент (исследует) -> Ты (принимаешь решение).
  * Жизненный цикл: Средний (Пока не найдена причина). Результат часто рождает новые задачи.

3. BRIEF (Концепт / Идея)
  * Цель: Проектирование нового. "Что хотим получить?".
  * Пользователь: Ты (описываешь хотелку) -> Агент (предлагает план) -> Ты (утверждаешь).
  * Жизненный цикл: Длинный. Это может стать основой для большой фичи.
